# Unit comparison for Empire Deluxe Combined Edition

Deployed application: http://edce.testingmagic.ovh/

## Description

Combat results between different types of units for [Empire Deluxe Combined Edition](https://killerbeesoftware.itch.io/empire-deluxe-combined-edition). The results were generated seperatelly with different application. The results can be sorted b win ratio and average lost HP cost. They can be filtered by unit type as well. The purpose for this app is to show what units are good or bad vs other unis. ï»¿

It is writen in [React](https://reactjs.org/) framework.
